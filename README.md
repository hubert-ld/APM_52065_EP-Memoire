# README 

This repository contains the code for the course project of statistical modelisation from Professor Rosenbaum at Ecole Polytechnique. We are looking for practical applications of the methods we have seen in class.

More specifically, we were interested in three main topics:
- `seine_et_marne` deals with **extreme quantile estimation** for the height of the Seine river. Then we also make a modelisation of how it couples with the Marne river, which is a tributary of the Seine, using the concept of **copulae**.
- `crypto` is a project about **time series**. We are trying to model the volatility of Bitcoin using an ARIMA model, and then we are trying to predict the future price of Bitcoin.
- `earthquakes` is also a project about **time series**. We try to apply the **Hawkes process** to model the occurrence of earthquakes in the world, and then we try to predict the future occurrence of earthquakes.
